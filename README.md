# @balintdorka.fit

Website for the @balintdorka.fit Pilates instructor brand.

## Files

```
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Sections

- Hero
- About
- Services (private, group, home sessions)
- Schedule
- Contact form

## Tech

- HTML5, CSS3, JavaScript
- Font Awesome icons
- Google Fonts (Playfair Display, Inter)
- Responsive layout with CSS Grid and Flexbox

## Deploying to GitHub Pages

1. Push the files to a public GitHub repository
2. Go to Settings > Pages
3. Set source to the main branch
4. The site will be live at `https://username.github.io/repo-name`

## Customization

Colors are defined as CSS variables in `styles.css`:

```css
:root {
    --primary-red: #e74c3c;
    --secondary-red: #c0392b;
    --primary-pink: #e91e63;
    --secondary-pink: #ad1457;
    --light-pink: #fce4ec;
}
```

Text content (name, schedule, contact info) can be edited directly in `index.html`.
